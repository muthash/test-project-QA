# app e2e tests

# For work with e2e tests you needed do next:

```shell
npm install
npx playwright install
```

# This is e2e commands

## If you want view how works test use this command

```bash
    npm run test:ui
```

#### other commands for tests

```bash
    npm run test:report
```

```bash
    npm run test:mobile-app
```
